##kali linux 渗透测试笔记一

说明：昨天成功的将网卡设置成为了monitor模式。所谓monitor模式。顾名思义，监听模式，网卡监听所有的数据包并将其发给系统。通过在监听模式下抓包看到了大量的无线广播包，而且抓到了手机微信的http的GET请求包。
如图：

下面叙述下设置monitor模式的步骤。

1.首先看网卡是否支持monitor模式。
```bash
iw list
```
这句用来查看网卡支持的类型。如下图：![2016050101.png](20160501/2016050101.png "")

iwconfig 的用法参见> http://blog.chinaunix.net/uid-24467128-id-3201533.html

2.如果支持的话，则首先应关闭网卡，然后设置为monitor模式。

我之前之所以设置不成功是因为没有先关闭网卡。

```bash
ifconfig 网卡名称 down
iwconfig 网卡名称 mode monitor
ifconfig 网卡名称 up
```

然后再利用wireshark抓包就可以收到其他主机的数据包了。
